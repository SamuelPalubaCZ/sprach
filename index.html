Help / FAQ

What is this?

This is a simulation of a numbers station, inspired by the East German "Stasi" speech generator. It converts numbers into a synthesized voice.

How do I use the Speech Generator?

Enter a numeric call sign and a message body of numbers, then press "Transmit Message". You can also click the number buttons or use the keyboard for manual input.

How does the Cipher Tool work?

You can encrypt text using the XOR cipher tool. The output is a string of numbers that you can then copy into the main message body for transmission. Use the same key to decrypt.

How do I record audio?

Use the audio recorder to record a message from your microphone. Press "Start Recording", and "Stop Recording" when you are finished. You can play back the recording using the audio player.
